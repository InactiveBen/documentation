## Welcome to Sweep

<Tooltip tip="A task is any resource that needs to be cleaned up, like connections, instances, or functions">
Sweep is a powerful and efficient task management and cleanup utility
</Tooltip> designed specifically for Roblox, with seamless <Tooltip tip="Knit is a popular framework for organizing Roblox code into services and controllers">Knit integration</Tooltip>. Our framework provides robust solutions for <Tooltip tip="Resource management involves tracking and cleaning up game objects, connections and other assets">resource management</Tooltip>, connection handling, and task cleanup to help you build more maintainable and performant games.
